## Overview

The data loading module handles acquisition and harmonization of cross-national and individual-level wellbeing data for f-factor analysis. Data sources include the World Happiness Report, OECD Better Life Index, World Bank Open Data, and MIDUS psychological wellbeing scales.

### parse_whr()

```python
def parse_whr(df: pd.DataFrame) -> pd.DataFrame
```

Renames WHR columns to PERMA+H schema and handles both cross-sectional and panel data formats.

<ParamField path="df" type="pd.DataFrame" required>
  Raw WHR dataframe from download_whr()
</ParamField>

<ParamField path="return" type="pd.DataFrame">
  Cleaned dataframe with standardized column names:
  - `P_positive_emotion` — Positive affect
  - `E_engagement_autonomy` — Freedom to make life choices
  - `R_relationships` — Social support
  - `M_meaning_life_sat` — Ladder score (Cantril 0-10)
  - `A_accomplishment_gdp` — Log GDP per capita
  - `H_health` — Healthy life expectancy
</ParamField>

**PERMA+H Mapping:**
- **P** (Positive emotion) → Positive affect
- **E** (Engagement) → Freedom to make life choices (autonomy proxy)
- **R** (Relationships) → Social support
- **M** (Meaning) → Ladder score (life evaluation)
- **A** (Accomplishment) → Log GDP per capita (material achievement)
- **H** (Health) → Healthy life expectancy

**Handles Two File Shapes:**
1. Cross-section: No year column, one row per country
2. Panel: Has year column; extracts most recent observation per country

## Individual-Level Analysis (MIDUS)

### load_midus()

```python
def load_midus(path: Path) -> pd.DataFrame
```

Loads MIDUS 2 (Midlife in the United States) individual-level psychological wellbeing data.

<ParamField path="path" type="Path" required>
  Path to MIDUS 2 DS0001 tab-delimited file (ICPSR 4652)
</ParamField>

<ParamField path="return" type="pd.DataFrame">
  Raw MIDUS dataframe with normalized column names
</ParamField>

**Data Source:**
- Study: MIDUS 2 Wave 2 (2004-2006)
- ICPSR Study Number: 4652
- N ≈ 4,963 respondents
- Variables: Ryff Psychological Wellbeing Scale (6 subscales), demographics, affect

**Ryff PWB Subscales:**
- `B1SPWBA1` — Autonomy (1-7 scale, mean of 3 items)
- `B1SPWBE1` — Environmental Mastery
- `B1SPWBG1` — Personal Growth
- `B1SPWBR1` — Positive Relations
- `B1SPWBU1` — Purpose in Life
- `B1SPWBS1` — Self-Acceptance

**Missing Value Handling:**
- MIDUS composites coded as 3-item *sums* (3-21 range)
- Missing coded as -1 or ≥ 90
- Automatically divided by 3 to convert to 1-7 mean scale

**File Location:** `04_individual_level_f_analysis.py:295`

### generate_synthetic_midus()

```python
def generate_synthetic_midus(n: int = 5000) -> pd.DataFrame
```

Generates synthetic individual-level data matching published MIDUS 2 psychometric properties.

<ParamField path="n" type="int" default="5000">
  Number of synthetic respondents to generate
</ParamField>

<ParamField path="return" type="pd.DataFrame">
  Synthetic dataset with same structure as real MIDUS data
</ParamField>

**Psychometric Benchmarks (from literature):**
- Inter-subscale correlations: 0.40–0.77
- Means: 4.80–5.53 on 1-7 scale
- SDs: 0.82–0.97
- Slight negative skew (ceiling effects)

**Generation Method:**
1. Cholesky decomposition of known correlation matrix
2. Multivariate normal sampling
3. Rescale to published means/SDs
4. Add ceiling effect correction
5. Clip to valid range [1, 7]

## Standardization

### StandardScaler Usage

All scripts use `sklearn.preprocessing.StandardScaler` for z-score normalization:

```python
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X_raw)
```

**Transformation:**
- Centers each variable to mean = 0
- Scales to standard deviation = 1
- Formula: `z = (x - μ) / σ`

**Why Standardize:**
1. Removes scale differences (e.g., 0-10 vs 1-7 scales)
2. Makes PCA loadings interpretable as correlations
3. Ensures equal weighting in factor analysis
4. Required for correlation-based covariance matrices

## Missing Data

### Strategy

All scripts use **listwise deletion** (complete-case analysis):

```python
df_clean = df.dropna(subset=PERMA_COLS)
```

**Rationale:**
- WHR/OECD data: Already country-level aggregates with minimal missingness
- MIDUS data: Ryff composites pre-computed; if missing, entire subscale unreliable
- Factor analysis requires complete correlation matrix

**Missingness Reporting:**

The `_report_missingness()` helper (script 01) prints per-variable counts:

```python
def _report_missingness(df: pd.DataFrame, cols: list, label: str = "")
```
