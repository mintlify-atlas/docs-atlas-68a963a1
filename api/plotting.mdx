The plotting module provides comprehensive visualization for f-factor analysis, including correlation heatmaps, PCA diagnostics, country rankings, mutualism dynamics, and resilience analysis.

## Saving Figures

All plotting functions return `plt.Figure` objects. Save with:

```python
fig = plot_correlation_matrix(df, cols)
fig.savefig("output_dir/figure_name.png")
```

**Recommended Settings:**
- DPI: 150 (already configured in rcParams)
- Format: PNG or PDF
- Bbox: "tight" (auto-configured)

**Output Directories:**
- Country-level: `wellbeing_figures/`
- Individual-level: `wellbeing_figures_individual/`
