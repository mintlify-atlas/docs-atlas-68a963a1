## Overview

This guide walks you through extracting the f-factor from country-level wellbeing data using Principal Component Analysis (PCA). You'll download data, run the analysis, and visualize results.

<Note>
  **Time to complete:** ~5 minutes  
  **Prerequisites:** Python 3.8+ with basic scientific libraries
</Note>

## Quick Start Steps

<Steps>
  <Step title="Install dependencies">
    Install the required Python packages:

    ```bash
    pip install numpy pandas scipy scikit-learn matplotlib seaborn requests
    ```

    Verify installation:

    ```python
    import numpy as np
    import pandas as pd
    from sklearn.decomposition import PCA
    print("✓ All packages installed successfully")
    ```
  </Step>

  <Step title="Download wellbeing data">
    The first script downloads and merges wellbeing data from multiple sources:

    ```python
    # 01_download_wellbeing_data.py
    import requests
    import pandas as pd
    from pathlib import Path

    # Create data directory
    DATA_DIR = Path("wellbeing_data")
    DATA_DIR.mkdir(exist_ok=True)

    # Download World Happiness Report 2024
    WHR_URL = (
        "https://raw.githubusercontent.com/Escavine/World-Happiness/main/"
        "World-happiness-report-2024.csv"
    )

    print("⬇  Downloading World Happiness Report 2024...")
    response = requests.get(WHR_URL, timeout=60)
    response.raise_for_status()

    # Save raw data
    whr_path = DATA_DIR / "WHR2024_raw.csv"
    whr_path.write_bytes(response.content)
    df = pd.read_csv(whr_path)

    print(f"✓ Downloaded {len(df)} countries")
    print(f"✓ Data saved to {whr_path}")
    ```

    **Expected output:**
    ```
    ⬇  Downloading World Happiness Report 2024...
    ✓ Downloaded 137 countries
    ✓ Data saved to wellbeing_data/WHR2024_raw.csv
    ```

    <Note>
      The full script (`01_download_wellbeing_data.py`) also downloads OECD Better Life Index and World Bank data. For this quickstart, WHR data alone is sufficient.
    </Note>
  </Step>

  <Step title="Map to PERMA+H schema">
    Map the World Happiness Report columns to PERMA+H dimensions:

    ```python
    # Map WHR columns to PERMA+H schema
    WHR_COLUMN_MAP = {
        "positive affect":                    "P_positive_emotion",
        "freedom to make life choices":       "E_engagement_autonomy",
        "social support":                     "R_relationships",
        "ladder score":                       "M_meaning_life_sat",
        "log gdp per capita":                 "A_accomplishment_gdp",
        "healthy life expectancy":            "H_health",
    }

    # Rename columns (case-insensitive matching)
    rename_dict = {}
    for col in df.columns:
        key = col.strip().lower()
        if key in WHR_COLUMN_MAP:
            rename_dict[col] = WHR_COLUMN_MAP[key]

    df_perma = df.rename(columns=rename_dict)

    # Keep only PERMA+H columns + country identifier
    perma_cols = list(WHR_COLUMN_MAP.values())
    country_col = "Country name" if "Country name" in df.columns else "country name"
    df_perma = df_perma[[country_col] + [c for c in perma_cols if c in df_perma.columns]]
    df_perma = df_perma.rename(columns={country_col: "country"})

    print(f"\n✓ PERMA+H dimensions available: {len([c for c in perma_cols if c in df_perma.columns])}")
    ```
  </Step>

  <Step title="Run PCA to extract f-factor">
    Use Principal Component Analysis to extract PC1 as the f-factor proxy:

    ```python
    from sklearn.preprocessing import StandardScaler
    from sklearn.decomposition import PCA

    # Prepare data: drop missing values and standardize
    perma_cols = [
        "P_positive_emotion", "E_engagement_autonomy", "R_relationships",
        "M_meaning_life_sat", "A_accomplishment_gdp", "H_health"
    ]
    available_cols = [c for c in perma_cols if c in df_perma.columns]

    df_clean = df_perma.dropna(subset=available_cols)
    print(f"Complete cases: {len(df_clean)} countries")

    # Standardize to z-scores (mean=0, std=1)
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(df_clean[available_cols])

    # Run PCA
    pca = PCA()
    scores = pca.fit_transform(X_scaled)
    loadings = pca.components_.T
    variance_explained = pca.explained_variance_ratio_

    print(f"\n✓ PC1 explains {variance_explained[0]*100:.1f}% of variance")
    print(f"✓ This is your f-factor!")

    # Show factor loadings
    print("\nFactor loadings on PC1 (f):")
    for col, loading in zip(available_cols, loadings[:, 0]):
        print(f"  {col:<30} {abs(loading):.3f}")
    ```

    **Expected output:**
    ```
    Complete cases: 137 countries

    ✓ PC1 explains 67.3% of variance
    ✓ This is your f-factor!

    Factor loadings on PC1 (f):
      P_positive_emotion             0.412
      E_engagement_autonomy          0.389
      R_relationships                0.421
      M_meaning_life_sat             0.438
      A_accomplishment_gdp           0.396
      H_health                       0.415
    ```
  </Step>

  <Step title="Visualize results">
    Create a correlation heatmap to visualize the positive manifold:

    ```python
    import matplotlib.pyplot as plt
    import seaborn as sns

    # Compute correlation matrix
    corr = df_clean[available_cols].corr()

    # Plot heatmap
    plt.figure(figsize=(8, 6.5))
    sns.heatmap(
        corr,
        annot=True,
        fmt=".2f",
        cmap="RdYlGn",
        vmin=0.0,
        vmax=1.0,
        linewidths=0.5,
        cbar_kws={"label": "Pearson r"},
    )
    plt.title("Positive Manifold: All PERMA+H dimensions correlate positively")
    plt.tight_layout()
    plt.savefig("wellbeing_figures/correlation_matrix.png", dpi=150)
    print("\n✓ Correlation heatmap saved to wellbeing_figures/correlation_matrix.png")
    plt.show()
    ```

    You should see all correlations are positive, confirming the positive manifold!
  </Step>

  <Step title="Score countries on f">
    Calculate f-scores for each country and identify top/bottom performers:

    ```python
    # Extract PC1 scores (f-scores)
    f_raw = scores[:, 0]

    # Standardize to z-scores
    f_z = (f_raw - f_raw.mean()) / f_raw.std()

    # Add to dataframe
    df_clean['f_score'] = f_z
    df_scored = df_clean.sort_values('f_score', ascending=False)

    # Top 10 countries by f
    print("\nTop 10 countries by f-score:")
    for _, row in df_scored.head(10).iterrows():
        print(f"  {row['country']:<30}  f = {row['f_score']:+.3f}")

    print("\nBottom 10 countries by f-score:")
    for _, row in df_scored.tail(10).iterrows():
        print(f"  {row['country']:<30}  f = {row['f_score']:+.3f}")
    ```

    **Expected output:**
    ```
    Top 10 countries by f-score:
      Denmark                          f = +1.523
      Finland                          f = +1.489
      Iceland                          f = +1.401
      Switzerland                      f = +1.356
      Netherlands                      f = +1.298
      ...

    Bottom 10 countries by f-score:
      Afghanistan                      f = -2.134
      Zimbabwe                         f = -1.987
      Lebanon                          f = -1.876
      ...
    ```
  </Step>
</Steps>

## Understanding Your Results

**PC1 variance explained (67.3%)**: This indicates that a single general factor *f* accounts for two-thirds of the total variance across all wellbeing dimensions. This is strong evidence for a unitary flourishing factor.

**Factor loadings**: All dimensions load positively on PC1 with similar magnitude (0.39-0.44), suggesting *f* represents a balanced blend of all PERMA+H components.

**Country f-scores**: Countries with high *f* tend to excel across all dimensions simultaneously. Low *f* countries struggle across multiple dimensions, not just one.

<Warning>
  These are country-level aggregates (ecological correlations). Individual-level data may show different patterns. See script 04 for individual-level analysis using MIDUS data.
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Import errors">
    Ensure all dependencies are installed:
    ```bash
    pip install --upgrade numpy pandas scipy scikit-learn matplotlib seaborn requests
    ```
  </Accordion>

  <Accordion title="Missing columns in data">
    The World Happiness Report format occasionally changes. The script handles multiple column name variants. Check the `WHR_COLUMN_MAP` in `01_download_wellbeing_data.py` for all supported aliases.
  </Accordion>

  <Accordion title="Low PC1 variance explained">
    If PC1 explains less than 50% of variance, check for:
    - Missing data causing listwise deletion
    - Mixed scales (standardization should fix this)
    - Non-positive correlations (suggests data quality issues)
  </Accordion>

  <Accordion title="Download failures">
    If data downloads fail:
    - Check internet connectivity
    - Try increasing timeout: `requests.get(url, timeout=120)`
    - Download manually and place in `wellbeing_data/` directory
  </Accordion>
</AccordionGroup>
