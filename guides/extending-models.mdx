## Custom Coupling Matrices

### Theory: Mutualism Model Dynamics

The mutualism system evolves according to:

```
dx_k/dt = a_k · x_k · (1 - x_k/C_k + Σ_{l≠k} M_kl · x_l / C_k)
```

Where **M_kl = coupling strength** between dimensions k and l.

### Default Calibration (Data-Driven)

By default, `03_models_full_f.py` estimates M from partial correlations:

```python
# From calibrate_mutualism_from_data() in 03_models_full_f.py:543
R = np.corrcoef(X_std.T)         # Zero-order correlations
P = inv(R)                        # Precision matrix
partial_corr = -P / np.outer(D, D)  # Partial correlations

# Scale to positive coupling range [0.05, 0.3]
M = np.abs(partial_corr)
M = 0.05 + 0.25 * (M - M.min()) / (M.max() - M.min() + 1e-8)
np.fill_diagonal(M, 0.0)
```

### Theory-Driven Custom Matrix

Replace with a theoretically-specified coupling structure:

```python
def build_custom_coupling_matrix(K: int) -> np.ndarray:
    """
    Example: Strong within-domain, weak between-domain coupling.
    
    Theory: PERMA+H has three clusters:
    - Hedonic: P, M (emotion + meaning)
    - Eudaimonic: E, A (engagement + accomplishment)
    - Relational: R, H (relationships + health)
    
    Hypothesis: Within-cluster coupling = 0.30, between-cluster = 0.10
    """
    M = np.full((K, K), 0.10)  # Baseline: weak coupling everywhere
    
    # Strong within-cluster coupling
    hedonic = [0, 3]           # P, M indices
    eudaimonic = [1, 4]        # E, A indices
    relational = [2, 5]        # R, H indices
    
    for cluster in [hedonic, eudaimonic, relational]:
        for i in cluster:
            for j in cluster:
                if i != j:
                    M[i, j] = 0.30
    
    np.fill_diagonal(M, 0.0)   # No self-coupling
    return M

# Usage in script 03
K = len(cols)
M_custom = build_custom_coupling_matrix(K)
sys = MutualismSystem(K, a=a, C=C, M=M_custom)
```

### Asymmetric Coupling (Advanced)

Test directional hypotheses (e.g., "Health supports Emotion more than vice versa"):

```python
def build_asymmetric_coupling(K: int) -> np.ndarray:
    """
    Example: Health → Positive Emotion coupling = 0.40
             Positive Emotion → Health coupling = 0.20
    """
    M = np.full((K, K), 0.15)  # Baseline symmetric
    
    # Indices: [P, E, R, M, A, H]
    P_idx, H_idx = 0, 5
    
    M[P_idx, H_idx] = 0.20  # P → H (emotion helps health, weak)
    M[H_idx, P_idx] = 0.40  # H → P (health enables emotion, strong)
    
    np.fill_diagonal(M, 0.0)
    return M
```

**Validation:** Check if asymmetric coupling better predicts empirical partial correlations:

```python
# Fit both models
sys_symmetric = MutualismSystem(K, a=a, C=C, M=M_symmetric)
sys_asymmetric = MutualismSystem(K, a=a, C=C, M=M_asymmetric)

# Simulate populations
eq_sym = sys_symmetric.simulate_population(n_individuals=200)
eq_asym = sys_asymmetric.simulate_population(n_individuals=200)

# Compare to empirical correlations
R_empirical = np.corrcoef(X_scaled.T)
R_sym = np.corrcoef(eq_sym.T)
R_asym = np.corrcoef(eq_asym.T)

# Frobenius norm (lower = better fit)
error_sym = np.linalg.norm(R_empirical - R_sym, 'fro')
error_asym = np.linalg.norm(R_empirical - R_asym, 'fro')

print(f"Symmetric model error: {error_sym:.3f}")
print(f"Asymmetric model error: {error_asym:.3f}")
```

## Modifying Threshold Logic (Model 4)

### Default: 25th Percentile Thresholds

From `03_models_full_f.py`:

```python
def compute_capabilities_adjusted_f(df, X_raw, f_scores, cols,
                                    threshold_quantile=0.25):
    thresholds = np.quantile(X_raw, threshold_quantile, axis=0)
    above_threshold = X_raw >= thresholds[np.newaxis, :]
    nussbaum_pass = above_threshold.all(axis=1).astype(float)
```

### Capability-Specific Absolute Thresholds

Nussbaum's capability approach uses **absolute** (not relative) thresholds:

```python
def compute_nussbaum_absolute_thresholds(X_raw, cols):
    """
    Theory-driven absolute thresholds for each PERMA+H dimension.
    
    Based on:
    - Nussbaum (2011): "Creating Capabilities"
    - Alkire & Santos (2014): MPI thresholds
    - Stiglitz-Sen-Fitoussi Commission recommendations
    """
    thresholds = {}
    
    # P: Positive emotion (0-10 affect scale)
    thresholds["P_positive_emotion"] = 5.0  # Below neutral = deprived
    
    # E: Engagement/autonomy (0-10 freedom scale)
    thresholds["E_engagement_autonomy"] = 6.0  # Basic self-direction
    
    # R: Relationships (0-10 social support scale)
    thresholds["R_relationships"] = 5.0  # Has at least one person to count on
    
    # M: Meaning (0-10 Cantril ladder)
    thresholds["M_meaning_life_sat"] = 5.0  # Midpoint = minimal acceptability
    
    # A: Accomplishment (log GDP per capita)
    thresholds["A_accomplishment_gdp"] = 8.5  # ~$5,000/year (basic needs)
    
    # H: Health (healthy life expectancy, years)
    thresholds["H_health"] = 60.0  # WHO global average
    
    # Convert dict to array in column order
    return np.array([thresholds[c] for c in cols])

# Replace in compute_capabilities_adjusted_f
thresholds = compute_nussbaum_absolute_thresholds(X_raw, cols)
```

### Multiple Deprivation Cutoffs (Alkire-Foster)

Default uses k=1/3 (deprived if failing ≥33% of dimensions). Test sensitivity:

```python
cutoffs = [0.2, 0.33, 0.5]  # 20%, 33%, 50% thresholds
results = {}

for k in cutoffs:
    af_cutoff = k
    identified = (deprivation_score >= af_cutoff).astype(float)
    censored_score = deprivation_score * identified
    
    H = identified.mean()
    A = censored_score[identified == 1].mean() if identified.sum() > 0 else 0
    M0 = H * A

# Plot sensitivity
import matplotlib.pyplot as plt
```

## Performance Optimization

### Speeding Up Bootstrap

Default: 1000 bootstrap resamples (~10 seconds)

**Parallel bootstrap:**
```python
from joblib import Parallel, delayed
from sklearn.decomposition import PCA

def bootstrap_single(X, ref_loadings, seed):
    np.random.seed(seed)
    n = X.shape[0]
    idx = np.random.choice(n, n, replace=True)
    pca_b = PCA(n_components=1)
    pca_b.fit(X[idx])
    l = pca_b.components_[0]
    if np.dot(l, ref_loadings) < 0:
        l = -l
    return l

def bootstrap_loadings_parallel(X, n_boot=1000, n_jobs=-1):
    ref_loadings = PCA(n_components=1).fit(X).components_[0]
    
    results = Parallel(n_jobs=n_jobs)(
        delayed(bootstrap_single)(X, ref_loadings, seed)
        for seed in range(n_boot)
    )
    
    return np.array(results)

# Replace in 02_analyze_wellbeing_f.py
boot_L = bootstrap_loadings_parallel(X, BOOTSTRAP_N, n_jobs=-1)
```

**Speedup:** ~5-10x faster on multi-core machines.
