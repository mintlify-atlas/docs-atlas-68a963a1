## Overview

Lyapunov stability analysis (Model 3) provides a rigorous mathematical framework for understanding the **resilience** and **recovery dynamics** of wellbeing systems. By analyzing the linearized dynamics around equilibrium, we can predict:

- Whether equilibrium is **stable** or **unstable**
- How quickly the system **recovers** from perturbations
- The **size of the basin of attraction** (how large a shock the system can withstand)
- **Early warning signals** of approaching instability (critical slowing down)

## Stability at Equilibrium

### Linear Stability Analysis

Near equilibrium x*, the dynamics are approximately linear:

```
dδx/dt ≈ J · δx
```

Where:
- `δx = x - x*` is the deviation from equilibrium
- `J` is the **Jacobian matrix** evaluated at x*

### Jacobian Matrix

For the mutualism system, the Jacobian is:

**Diagonal elements (k = l):**
```
J_kk = a_k · (1 - 2x*_k/C_k + Σ_{l≠k} M_kl · x*_l / C_k)
```

**Off-diagonal elements (k ≠ l):**
```
J_kl = a_k · x*_k · M_kl / C_k
```

<Note>
The Jacobian encodes both self-regulation (diagonal) and mutual support (off-diagonal).
</Note>

## Eigenvalue Interpretation

### Stability Criterion

An equilibrium is **asymptotically stable** if and only if:

```
Re(λ_i) < 0  for all eigenvalues λ_i of J
```

If any eigenvalue has positive real part, the equilibrium is **unstable**.

### Dominant Eigenvalue

The **dominant eigenvalue** λ₁ is the one with the largest (least negative) real part:

```
λ₁ = max{Re(λ_i)}
```

This eigenvalue controls the **slowest relaxation mode** and determines recovery time.

### Physical Interpretation

- **Large negative Re(λ₁) (e.g., -0.5)**: Fast recovery, strong stability
- **Small negative Re(λ₁) (e.g., -0.05)**: Slow recovery, critical slowing down
- **Re(λ₁) = 0**: Bifurcation point, loss of stability
- **Positive Re(λ₁)**: Unstable, exponential divergence

## Return Time (τ)

### Definition

The **characteristic return time** is:

```
τ = 1 / |Re(λ₁)|
```

This is the time scale for perturbations to decay by a factor of e ≈ 2.718.

### Interpretation

<ParamField path="τ = 10-20" type="moderate recovery">

</ParamField>

### Critical Slowing Down

As a system approaches a bifurcation (tipping point):
- λ₁ → 0
- τ → ∞
- Recovery slows dramatically
- **Early warning signal** of regime shift

This phenomenon is observed in ecological systems, financial markets, and potentially wellbeing dynamics.

## Basin of Attraction

### Lyapunov Equation

To quantify the basin of attraction, solve the continuous-time Lyapunov equation:

```
J' P + P J = -Q
```

Where:
- `J` = Jacobian at equilibrium
- `Q` = positive-definite matrix (typically Q = I, identity)
- `P` = solution matrix (positive-definite if system is stable)

### Basin Ellipsoid

The solution P defines an ellipsoidal basin:

```
B = {δx : δx' P δx ≤ c}
```

Perturbations within this ellipsoid return to equilibrium.

### Resilience Metric

The **resilience** is approximated as:

```
R(x*) = 1 / sqrt(λ_max(P))
```

Where λ_max(P) is the largest eigenvalue of P.

**Interpretation:**
- **High R**: Large basin, can withstand large perturbations
- **Low R**: Small basin, fragile to shocks

This is the **minimum perturbation size** (in the most vulnerable direction) that escapes the basin.

## Perturbation-Response Analysis

### Impulse Response

To test recovery dynamics:

1. Start at equilibrium x*
2. Apply perturbation: x(0) = x* + δx
3. Simulate forward: solve dx/dt = F(x)
4. Measure deviation: ||x(t) - x*||

Expected decay:
```
||x(t) - x*|| ≈ ||δx|| · exp(Re(λ₁) · t)
```

### Typical Perturbations

- **Single domain shock**: δx = [0, 0, -0.3·x*_k, 0, 0, 0]
- **Uniform shock**: δx = -0.2·x* (all domains down 20%)
- **Random shock**: δx sampled from multivariate normal

## Stable vs Unstable Equilibria

### Stable Equilibrium

**Characteristics:**
- All Re(λ_i) < 0
- Small perturbations decay exponentially
- System returns to x* from nearby states
- Basin of attraction has positive volume
- **Resilient** to shocks

**Example:**
```
Eigenvalues: [-0.25, -0.18, -0.12, -0.08, -0.06, -0.03]
τ = 33.3 (moderate return time)
Resilience = 0.15 (moderate basin)
```

### Unstable Equilibrium

**Characteristics:**
- At least one Re(λ_i) > 0
- Perturbations grow exponentially
- System moves away from x*
- Basin of attraction has zero volume
- **Not resilient** — cannot be observed

**Example:**
```
Eigenvalues: [+0.05, -0.15, -0.12, -0.08, -0.06, -0.03]
Unstable — cannot persist
```

<Warning>
Unstable equilibria are **mathematical solutions** but not **observable states**. Real systems near unstable equilibria exhibit runaway dynamics.
</Warning>

## Mathematical Details

### Exponential Stability

For stable equilibrium with dominant eigenvalue λ₁:

```
||x(t) - x*|| ≤ M · ||x(0) - x*|| · exp(Re(λ₁) · t)
```

Where M is a constant depending on eigenvector geometry.

### Lyapunov Function

The solution P to the Lyapunov equation defines a Lyapunov function:

```
V(x) = (x - x*)' P (x - x*)
```

Properties:
- V(x*) = 0
- V(x) > 0 for x ≠ x*
- dV/dt = -||x - x*||² < 0 (along trajectories)

This proves x* is asymptotically stable.

### Sensitivity to M

The Jacobian (and hence eigenvalues) depend on coupling matrix M:

```
∂λ₁/∂M_kl ≈ eigenvector sensitivity
```

Increasing coupling generally:
- Increases off-diagonal J elements (positive feedback)
- Can destabilize if coupling too strong

## Limitations

### Linear Approximation

Lyapunov analysis uses **local linearization**. Predictions hold for small perturbations only.

For large shocks, nonlinear effects matter:
- Actual recovery may be faster or slower than predicted
- Basin boundary may be irregular (not ellipsoidal)

### Cross-Sectional Data

Calibrating from cross-sectional data assumes:
- Countries are at equilibrium
- Same coupling matrix M for all
- Growth rates a not identifiable

Longitudinal data would enable better parameter estimation.
