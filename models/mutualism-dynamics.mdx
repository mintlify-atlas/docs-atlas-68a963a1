## Overview

The mutualism dynamics model (Model 2) provides a **generative mechanism** for the positive manifold observed in wellbeing data. Unlike the bifactor CFA which assumes a latent general factor, mutualism shows how **direct mutual support** between domains produces correlated growth patterns without any hidden common cause.

<Note>
This is a crucial theoretical insight: the positive manifold in wellbeing data does not require a latent variable explanation. It can emerge naturally from mutual reinforcement dynamics.
</Note>

## The Mutualism Equations

### Coupled Logistic Growth

Each wellbeing domain k evolves according to:

```
dx_k/dt = a_k · x_k · (1 - x_k/C_k + Σ_{l≠k} M_kl · x_l / C_k) + σ_k · ξ_k(t)
```

**Parameters:**

<ParamField path="x_k" type="float">
  Current level of wellbeing domain k (e.g., positive emotion, relationships)
</ParamField>

<ParamField path="a_k" type="float">
  Intrinsic growth rate for domain k (typical value: 0.5)
</ParamField>

<ParamField path="C_k" type="float">
  Carrying capacity — baseline level achievable without mutual support
</ParamField>

<ParamField path="M_kl" type="float">
  Mutualistic coupling strength from domain l to domain k (M_kk = 0)
</ParamField>

<ParamField path="σ_k" type="float">
  Noise amplitude for stochastic simulations (optional)
</ParamField>

<ParamField path="ξ_k(t)" type="float">
  White noise process for stochastic version
</ParamField>

### Key Insight: Mutualism Term

The term `Σ_{l≠k} M_kl · x_l / C_k` represents **mutual support**:
- When other domains (l) are high, they **boost** the growth of domain k
- This creates **positive feedback loops** between domains
- The coupling matrix M encodes which domains support which others

## Equilibrium Analysis

### Non-Trivial Equilibrium

At equilibrium (dx_k/dt = 0, x_k > 0), the growth equation reduces to:

```
1 - x_k/C_k + Σ_{l≠k} M_kl · x_l / C_k = 0
```

Rearranging:
```
x_k = C_k + Σ_{l≠k} M_kl · x_l
```

### Matrix Form Solution

Define M̃ where M̃_kl = M_kl for k≠l and 0 on diagonal. Then:

```
(I - M̃) x* = C
x* = (I - M̃)⁻¹ C
```

**Critical result:** When M has positive entries, `x* > C` for all domains. **Mutualism lifts equilibrium above carrying capacity.**

### Why This Matters

Individuals with higher baseline capacities (larger C) will have higher equilibria across all domains. But the **pattern of correlations** comes from the coupling matrix M, not from a shared latent variable.

## Positive Manifold Without Latent Variables

### Population Simulation

The model demonstrates that a positive manifold emerges when:

1. **Each individual** has slightly different parameters (C_i, a_i, M_i)
2. **Environmental factors** create correlated perturbations to carrying capacities
3. **Mutualistic dynamics** amplify these correlations

Result: Population covariance matrix shows all-positive correlations and a dominant first principal component, **even though no latent variable generated the data**.

### Mechanism

```
Individual differences in C → Different equilibria x* → Population covariance
                                      ↑
                              Amplified by M (coupling)
```

The coupling matrix M acts as a **multiplier** that amplifies small differences in baseline capacities into larger differences in equilibrium outcomes.

#### jacobian_at_equilibrium()

```python
def jacobian_at_equilibrium(self, x_star):
    """
    Compute the Jacobian matrix J at equilibrium x*.
    
    J_kl = ∂ẋ_k / ∂x_l evaluated at x*.
    """
```

For k = l (diagonal):
```
J_kk = a_k * (1 - 2x*_k/C_k + Σ_{l≠k} M_kl x*_l / C_k)
```

For k ≠ l (off-diagonal):
```
J_kl = a_k * x*_k * M_kl / C_k
```

<Note>
The Jacobian is essential for stability analysis (see [Lyapunov Stability](/models/lyapunov-stability)).

**Calibration strategy:**

1. **Carrying capacities (C_k)**: Set to 70% of observed means
   - Rationale: Mutualism lifts equilibrium above baseline, so x* > C

2. **Coupling matrix (M_kl)**: Derived from partial correlations
   - Partial correlations proxy **direct** interactions after removing indirect paths
   - Scaled to range [0.05, 0.3]

3. **Growth rates (a_k)**: Fixed at 0.5
   - Not identifiable from cross-sectional data
   - Affects time scale but not equilibrium values

## Trajectory Convergence

The deterministic mutualism system exhibits **asymptotic stability**:

- Starting below equilibrium (x_0 < x*) → domains **grow** toward x*
- Starting above equilibrium (x_0 > x*) → domains **decline** toward x*
- Convergence is **monotonic** when coupling is moderate

### Visualization

Trajectory plots show:
- **Dashed lines**: Carrying capacities C_k
- **Dotted lines**: Equilibrium x*_k
- **Solid curves**: Time evolution x_k(t)

Key observation: All trajectories converge to x* > C, demonstrating mutualistic lift.

## Mean Coupling Strength

```python
def mean_coupling_strength(self):
    """M̄ = 2/(K(K-1)) · Σ_{k<l} M_kl"""
```

Average coupling over all unique pairs. This scalar summarizes interaction strength:
- **Low M̄ (< 0.1)**: Weak mutualism, near-independent domains
- **Moderate M̄ (0.1-0.3)**: Optimal regime with strong f but stable dynamics
- **High M̄ (> 0.4)**: Risk of instability or runaway cascades

## Key Predictions

### 1. Positive Manifold
All pairwise correlations positive when M has positive entries.

### 2. Dominant PC1
First principal component explains large variance (40-60%) even without latent variable.

### 3. Correlation with f
Higher mean coupling M̄ predicts stronger first principal component.

### 4. Environmental Amplification
Shared environmental perturbations to C amplified by mutual support dynamics.

## Comparison with Bifactor CFA

| Aspect | Bifactor CFA | Mutualism Model |
|--------|-------------|----------------|
| **Nature** | Latent variable model | Dynamical process model |
| **Mechanism** | Common cause (f) | Mutual reinforcement |
| **Time** | Cross-sectional | Dynamic (with equilibrium) |
| **Interpretation** | f as latent trait | f as emergent pattern |
| **Parameters** | Loadings λ, uniqueness θ | Growth rates a, coupling M, capacities C |

Both models can produce similar covariance structures, but they have different causal interpretations.
