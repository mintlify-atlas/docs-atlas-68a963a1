The first step in the f-factor analysis pipeline downloads and harmonizes wellbeing data from three public sources, mapping them to the **PERMA+H** framework for flourishing.

## Overview

Script: `01_download_wellbeing_data.py`

This script retrieves country-level wellbeing indicators and harmonizes them into a unified dataset suitable for extracting the general factor of flourishing (f).

## Data Sources

### 1. World Happiness Report 2024 (WHR)

Primary data source from Gallup World Poll, accessed via GitHub mirror:

```python
WHR_URL = (
    "https://raw.githubusercontent.com/Escavine/World-Happiness/main/"
    "World-happiness-report-2024.csv"
)
```

**Available indicators:**
- Positive affect
- Freedom to make life choices
- Social support
- Ladder score (Cantril 0-10 life evaluation)
- Log GDP per capita
- Healthy life expectancy
- Generosity

## PERMA+H Mapping

The script maps World Happiness Report columns to the PERMA+H theoretical framework:

```python
WHR_COLUMN_MAP = {
    "positive affect":                    "P_positive_emotion",
    "freedom to make life choices":       "E_engagement_autonomy",
    "social support":                     "R_relationships",
    "ladder score":                       "M_meaning_life_sat",
    "log gdp per capita":                 "A_accomplishment_gdp",
    "healthy life expectancy":            "H_health",
    # Reference columns
    "negative affect":                    "ref_negative_affect",
    "generosity":                         "ref_generosity",
    "perceptions of corruption":          "ref_corruption",
}
```

| **Domain** | **PERMA+H Dimension** | **WHR Proxy** |
|------------|------------------------|---------------|
| **P** | Positive emotion | Positive affect |
| **E** | Engagement | Freedom to make life choices (autonomy) |
| **R** | Relationships | Social support |
| **M** | Meaning | Ladder score (life satisfaction) |
| **A** | Accomplishment | Log GDP per capita (material achievement) |
| **H** | Health | Healthy life expectancy |

<Warning>
  The WHR data are **country-level averages** (ecological correlations), not individual survey responses. The positive manifold observed at this level may differ from individual-level structure.
</Warning>

## Download Workflow

<Steps>
  <Step title="Download WHR 2024 data">
    Fetch the World Happiness Report CSV from GitHub and save to `wellbeing_data/WHR2024_raw.csv`.
    
    ```python
    def download_whr() -> pd.DataFrame:
        print("⬇  World Happiness Report 2024 …", end=" ", flush=True)
        r = requests.get(WHR_URL, timeout=60, headers=HEADERS)
        r.raise_for_status()
        path = DATA_DIR / "WHR2024_raw.csv"
        path.write_bytes(r.content)
        df = pd.read_csv(path)
        print(f"done. ({len(df)} rows, {len(df.columns)} cols)")
        return df
    ```
  </Step>

  <Step title="Parse and rename WHR columns">
    Apply PERMA+H schema mapping, handling both cross-sectional and panel formats. If a year column is present, select the most recent observation per country.
    
    ```python
    def parse_whr(df: pd.DataFrame) -> pd.DataFrame:
        # Identify country column (flexible matching)
        country_col = next(
            (c for c in COUNTRY_COL_CANDIDATES if c in df.columns), None
        )
        
        # Subset to most recent year if panel
        if year_col:
            df_cs = (
                df.sort_values(year_col)
                  .groupby(country_col, as_index=False)
                  .last()
            )
        
        # Rename columns to PERMA+H schema
        rename = {}
        for c in df_cs.columns:
            key = c.strip().lower()
            if key in WHR_COLUMN_MAP:
                rename[c] = WHR_COLUMN_MAP[key]
        df_cs = df_cs.rename(columns=rename)
        return df_cs
    ```
  </Step>

  <Step title="Download OECD BLI data">
    Retrieve the Better Life Index CSV and pivot from long to wide format (one column per indicator).
    
    ```python
    def parse_oecd_bli(df: pd.DataFrame) -> pd.DataFrame:

            .groupby([country_col, "_label"])[value_col]

        )
        return df_wide
    ```
  </Step>

  <Step title="Fetch World Bank indicators">
    Query the World Bank API for supplemental health and economic data (most recent value per country).
    
    ```python
    def _fetch_wb_indicator(indicator: str, label: str) -> pd.DataFrame:
        url = f"{WB_API}/{indicator}?format=json&per_page=300&mrv=1"
        r = requests.get(url, timeout=30, headers=HEADERS)
        payload = r.json()
        records = payload[1] if len(payload) > 1 else []
        rows = [
            {
                "country": rec["country"]["value"],
                "iso3":    rec["countryiso3code"],
                label:     rec["value"],
            }
            for rec in records if rec.get("value") is not None
        ]
        return pd.DataFrame(rows)
    ```
  </Step>

  <Step title="Merge all sources">
    Join WHR, OECD BLI, and World Bank data using fuzzy country name matching (lowercase + strip).
    
    ```python
    def merge_sources(whr, bli, wb) -> pd.DataFrame:
        merged = whr.copy()
        merged["_key"] = merged["country"].str.lower().str.strip()
        
        if not bli.empty:
            bli["_key"] = bli["country_oecd"].str.lower().str.strip()
            merged = pd.merge(merged, bli, on="_key", how="left")
        
        if not wb.empty:
            wb["_key"] = wb["country"].str.lower().str.strip()
            merged = pd.merge(merged, wb, on="_key", how="left")
        
        return merged.drop(columns=["_key"])
    ```
  </Step>

  <Step title="Save merged dataset">
    Write the harmonized dataset to `wellbeing_data/wellbeing_merged.csv` for use in the analysis script.
    
    The script reports:
    - Final shape (N countries × M variables)
    - Missingness in PERMA+H columns
    - Summary statistics (mean, std, skew)
    - Complete case count (all 6 PERMA+H dimensions present)
  </Step>
</Steps>
